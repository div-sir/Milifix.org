# A320 飛行手冊

## ✅ A320neo V2 – Full Startup to Takeoff Checklist

### 🔌 Cold & Dark Startup

---

- [ ]  BAT 1 & BAT 2 → ON
- [ ]  EXT PWR → ON (if available)
- [ ]  Crew Supply → ON
- [ ]  NAV & LOGO Lights → ON (as needed)
- [ ]  ADIRS (3 knobs) → NAV (Align IRS)
- [ ]  Set Brightness for ECAM & MCDU

### 🧭 MCDU Initialization & Flight PlanINIT A page → Enter FROM/TO ICAO

---

- [ ]  Flight number / Cost Index / CRZ FL
- [ ]  F-PLN page → Insert SID, route, STAR
- [ ]  INIT B page → Set ZFW, Block Fuel
- [ ]  PERF TO page → Set V1 / VR / V2
- [ ]  Set FLEX TEMP or TOGA
- [ ]  Set Flaps configuration (usually Flaps 1+F)

### 🔧 Overhead & Systems Setup

---

- [ ]  Fuel Pumps → ON
- [ ]  APU Master → ON
- [ ]  APU Start → Wait for APU AVAIL
- [ ]  APU Bleed → ON
- [ ]  Seatbelt & No Smoking Signs → ON
- [ ]  EFIS Baro/STD → Set QNH
- [ ]  Autobrake → MAX
- [ ]  Transponder → Code + Set to AUTO

### 🛻 Pushback & Engine Start

---

- [ ]  Beacon Light → ON
- [ ]  Call Pushback Tug
- [ ]  ENG MODE Selector → IGN/START
- [ ]  Start ENG 2 → Monitor N2, fuel flow, EGT
- [ ]  Start ENG 1 → Monitor
- [ ]  ENG MODE Selector → NORM
- [ ]  APU Bleed → OFF (if engines stabilized)
- [ ]  APU Master → OFF (optional)

### 🛫 Before Taxi

---

- [ ]  Check FLAPS → Set as per PERF (1+F)
- [ ]  SPOILERS → ARM
- [ ]  PITCH TRIM → Set (e.g., UP 0.8)
- [ ]  Rudder Trim → NEUTRAL
- [ ]  ECAM Status → Check
- [ ]  ND Range & Mode → Set
- [ ]  TAXI Light → ON
- [ ]  Parking Brake → RELEASE

### ✈️ Taxi Checklist

---

- [ ]  Check Brakes & Nosewheel Steering
- [ ]  Perform flight control check
- [ ]  Check flight instruments (PFD, ND)
- [ ]  Set TO CONFIG button → “TO CONFIG NORMAL”
- [ ]  Contact ATC for clearance

### 🛬 Before Takeoff (Final Items)

---

- [ ]  Takeoff RWY Confirmed & Aligned
- [ ]  ECAM → Takeoff Config NORMAL
- [ ]  RWY Turnoff & Strobe Lights → ON
- [ ]  Landing Lights → ON
- [ ]  TCAS → TA/RA

### 🚀 Takeoff Roll

- [ ]  Thrust Levers → FLEX or TOGA detent
- [ ]  Monitor FMA: “SRS | RWY | CLB”
- [ ]  At V1 → Continue
- [ ]  At VR → Rotate to ~15° pitch
- [ ]  Positive Climb → Gear UP
- [ ]  At Acceleration Altitude → Lower pitch, retract flaps
- [ ]  Set CLB detent
- [ ]  AP1 → ON (if desired)

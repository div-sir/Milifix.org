# F35 飛行手冊

## ✅ F-35 起飛前檢查清單（F-35 Pre-Flight Checklist）

> 適用於 A/B/C 型，支援 STOVL 模式（B型）
> 

### 📦 初始化 & 電源流程 (Initialization & Power)

---

- [ ]  IPP 主電源開啟（IPP Master Switch ON）
- [ ]  ICC1 & ICC2 電源開啟（ICC1 & ICC2 Switches ON）
- [ ]  電池開啟（Battery Switch ON）
- [ ]  IPP 設定為 AUTO（IPP Selector AUTO）
- [ ]  主控制螢幕是否亮起？（PCD ON）
- [ ]  檢查油量與主油閥（FUEL Page & MSFOV → OPEN）

### ⚙️ 引擎啟動 (Engine Start)

---

- [ ]  引擎開關設為 NORM（Engine Switch → NORM）
- [ ]  IPP 啟動（IPP Selector → START）
- [ ]  推油門至 IDLE（Throttle → IDLE）
- [ ]  引擎穩定顯示正常（Check PCD ENG Page）
- [ ]  發電機警告消失（Verify GENERATOR LOW → OFF）

### 🛫 起飛前設定 (Before Takeoff)

---

- [ ]  座艙蓋關閉並鎖定（Canopy CLOSED & LOCKED）
- [ ]  彈射座椅 ARM（Ejection Seat → ARMED）
- [ ]  剎車檢查完成（Check Brakes）
- [ ]  配平設定（Trim）→ 依重量設定 7~10（Elev Trim ~ 7-10）
- [ ]  確認起落架放下（Gear → DOWN）
- [ ]  起飛燈 / 滑行燈開啟（LAND/TAXI LIGHTS → ON）
- [ ]  確認 IFF / NAV / COM 設定完成（IFF, NAV, COMs → SET）

### 🚀 一般起飛流程（Normal Takeoff - A/B/C）

---

- [ ]  排隊滑行至跑道（Taxi to Runway）
- [ ]  踩住剎車（Hold Brakes）
- [ ]  推油門至 80%（Set Throttle ~80%）
- [ ]  放開剎車（Release Brakes）
- [ ]  全油門至 MRT（Max Rated Thrust ~105%）
- [ ]  抬頭（Rotate）：
    - A/B：150~170 KIAS 抬頭（Rotate at 150–170 KIAS）
    - C：130 KIAS 抬頭（Rotate at ~130 KIAS）
- [ ]  正爬升時收起起落架（Gear UP when positive climb）

### 🛩 F-35B 短場起飛（STOVL Mode - Short Takeoff）

---

- [ ]  啟動 STOVL 模式（Press STOVL Button）
- [ ]  驗證 STOVL 模式啟動（FCS Page → STOVL Mode = ON）
- [ ]  踩住剎車（Hold Brakes）
- [ ]  推油門至 80%（Throttle ~ 80%）
- [ ]  放開剎車（Release Brakes）
- [ ]  全推力 MRT（Throttle → MRT ~105%）
- [ ]  80 KIAS 抬頭（Rotate at ~80 KIAS）
- [ ]  300 ft AGL 收起落架（Gear UP above 300 ft）
- [ ]  500 ft AGL 離開 STOVL 模式（Exit STOVL above 500 ft）
- [ ]  驗證進入 EXER 模式（Check FCS → EXER Mode）

### 🎮 控制提示：STOVL 模式下操作方式（STOVL Control Tips）

- [ ]  上下飛行：升降舵軸（Elevator Axis）
- [ ]  左右橫移：副翼軸（Aileron Axis）
- [ ]  左右轉向：方向舵軸（Rudder Axis）
- [ ]  前進/後退速度：副翼配平（Aileron Trim）控制速度

## 🛸 F-35B 垂直起降操作清單（F-35B Vertical Takeoff & Landing Checklist）

> Hover Mode 適用於 F-35B 型號，起飛重量需 低於 40,600 lbs
> 

---

### ✈️ 垂直起飛（Vertical Takeoff / VTOL）

---

- [ ]  定位於垂直起飛區域（Line up on takeoff pad or area）
- [ ]  啟用 STOVL 模式（Press STOVL Button）
✅ FCS Page 顯示 STOVL → ON
- [ ]  啟用 Hover 模式（Press HOVER Button or use FCS Page）
✅ FCS Page 顯示 HOVER → ON
- [ ]  檢查推力與姿態控制（Check thrust axis and stability）
- [ ]  小油門上升 → 懸停至所需高度（Throttle slowly → HOVER to desired altitude）
- [ ]  轉為前進飛行準備（Prepare for forward flight）：
    - [ ]  離開 HOVER 模式（Exit HOVER Mode）
    - [ ]  推油門至 MRT（Max Rated Thrust ~105%）
    - [ ]  加速滑行
    - [ ]  300 ft AGL → 收起落架（Gear UP above 300 ft）
    - [ ]  500 ft AGL → 離開 STOVL 模式（Exit STOVL Mode）
    - [ ]  驗證 EXER 模式啟用（Verify FCS Page shows EXER Mode）

### 🛬 垂直降落（Vertical Landing / VSTOL）

- [ ]  減速至 < 200 KIAS（Slow down below 200 KIAS）
- [ ]  啟用 STOVL 模式（Press STOVL Button）
✅ FCS Page 顯示 STOVL → ON
- [ ]  低於 200 ft AGL 啟用 HOVER 模式（Below ~200 ft → Enter HOVER Mode）
- [ ]  使用：
    - 升降舵 → 控制上下
    - 副翼 → 左右橫移
    - 方向舵 → 水平轉向
    - 副翼配平 → 控制前進/後退速度
- [ ]  穩定懸停於降落點上方（Hover over landing zone）
- [ ]  緩慢下降（Gently descend）
- [ ]  接地後油門退至 IDLE（Throttle to IDLE after touchdown）
- [ ]  關閉 HOVER → STOVL 模式（Exit HOVER and then STOVL Mode）

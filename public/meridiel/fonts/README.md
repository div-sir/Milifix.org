# Meridiel fonts

These WOFF2 files are the Latin subsets from Fontsource 5.2.6 packages:

- `@fontsource/jost@5.2.6`
- `@fontsource/space-mono@5.2.6`
- `@fontsource/dm-serif-display@5.2.6`

The original font families are distributed under the SIL Open Font License
1.1. See `LICENSE.txt`. The files are served from Meridiel's own origin so the
initial page does not depend on Google Fonts.
